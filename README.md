# HR Employee Attrition & Workforce Analysis

## About the Project

This project focuses on analyzing employee attrition and workforce trends using HR data. The objective was to identify the factors that influence employee turnover and build an interactive Power BI dashboard to help HR teams make data-driven decisions.

The project combines Python for data preparation and analysis with Power BI for creating an interactive dashboard.

## Dashboard Preview

![Dashboard](dashboard.png)

---

## Project Objectives

- Analyze employee attrition across departments and job levels.
- Compare average monthly income between departments.
- Study the relationship between overtime and attrition.
- Understand employee job satisfaction and work-life balance.
- Build an interactive HR dashboard for workforce analysis.

---

## Tools Used

- Power BI
- Python (Pandas, Jupyter Notebook)
- Microsoft Excel

---

## Dataset

The dataset contains HR records for 10,000 employees, including:

- Employee demographics
- Department and Job Level
- Monthly Income
- Years at Company
- Job Satisfaction
- Work-Life Balance
- Overtime
- Performance Rating
- Attrition

---

## Dashboard Features

The dashboard includes:

- Employee Overview
- Attrition Analysis
- Department-wise Salary Analysis
- Overtime vs Attrition
- Job Satisfaction Distribution
- Work-Life Balance Distribution
- Interactive slicers for Department, Job Level and Overtime
